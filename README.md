# nodejs-cli-rabbitmq-cluster

## Description
A cluster producer/consumer
message broker demo.

## Tech stack
- nodejs

## Docker stack
- node:latest
- harbur/rabbitmq-cluster

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
